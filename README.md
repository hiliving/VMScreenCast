VMScreenCast
========

使用 Cling 实现了 DMS DMC 功能的投屏应用

### #简介
本项目使用 [4thline/cling](https://github.com/4thline/cling) 库实现投屏功能，其中实现了`DMS`及`DMC`功能

### #实现功能
- 浏览本地资源投屏播放
- 模拟请求网络资源，投屏播放网络视频
- 控制投屏设备，静音，拖动，播放，暂停等

### #已知问题
- 因为是用另一台手机安装 AirPin 测试的，现在测试`设置音量`和`视频的拖动进度`不可用
